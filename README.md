## How to Run

1. Open the project in **DartPad** (https://dartpad.dev/flutter) or in a **Flutter environment**.  
2. Make sure the `lib/main.dart` file is present.  
3. Run the app to see the workflow:
   - Restaurant list  
   - Menu selection  
   - Checkout / order confirmation
